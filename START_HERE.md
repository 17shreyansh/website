# 🎯 START HERE

## Welcome to RSD Bharti Industries Website! 🎉

This is a **premium construction materials website** built with Next.js, React, and Tailwind CSS.

---

## ⚡ Quick Start (Choose One)

### Option 1: Windows Users (Easiest)
```
Double-click: setup.bat
```
That's it! The website will open automatically.

### Option 2: Mac/Linux Users
```bash
npm install
npm run dev
```
Then open: http://localhost:3000

### Option 3: Already Installed?
```bash
npm run dev
```
