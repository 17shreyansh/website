# RSD Bharti Industries - Premium Construction Materials Website

A modern, premium website for RSD Bharti Industries Pvt. Ltd., showcasing 25 years of excellence in construction materials supply.

## 🎨 Design Features

- **Premium Brand Identity**: Royal purple and gold color scheme
- **Modern Tech Stack**: Next.js 14, React, TypeScript, Tailwind CSS
- **Smooth Animations**: Framer Motion for elegant transitions
- **Fully Responsive**: Mobile-first design approach
- **Component-Based**: Reusable, maintainable code structure

## 🚀 Tech Stack

- **Framework**: Next.js 14 (App Router)
- **Language**: TypeScript
- **Styling**: Tailwind CSS
- **Animations**: Framer Motion
- **Icons**: Lucide React
- **Font**: Inter (Google Fonts)

## 📦 Installation

1. **Install Dependencies**
   ```bash
   npm install
   ```

2. **Run Development Server**
   ```bash
   npm run dev
   ```

3. **Open Browser**
   Navigate to [http://localhost:3000](http://localhost:3000)

## 📄 Pages

### Home Page (`/`)
- Hero section with company tagline
- Trust statistics (25+ years, 12+ brands, 1000+ projects)
- About preview with core values
- Materials showcase
- Mobile app promotion
- Call-to-action sections

### About Page (`/about`)
- Company story and history
- Mission and vision statements
- 25-year timeline
- Core values
- Leadership message

### Materials Page (`/materials`)
- Premium Cement brands (UltraTech, Dalmia, ACC, Ambuja, Shree, Birla)
- TMT Steel brands (Tata Tiscon, JSW, Jindal, Shyam, Kamdhenu)

- Contact form
- Phone, email, address details
- Business hours
- Map placeholder
- Multiple contact methods
