## ✨ Animation Effects

### Hover States
```
Button:     [Normal] → [Lifted + Shadow]
Card:       [Flat]   → [Raised -8px]
Link:       [Gray]   → [Purple]
Icon:       [Static] → [Colored]
```

### Scroll Animations
```
Element enters viewport:
  Opacity: 0 → 1
  Y-position: +30px → 0
  Duration: 0.8s
  Easing: ease-out
```

### Mobile Menu
```
Closed: [☰]
  ↓ Click
Open:  [✕]
  Menu slides down
  Backdrop appears
  Links fade in
```
